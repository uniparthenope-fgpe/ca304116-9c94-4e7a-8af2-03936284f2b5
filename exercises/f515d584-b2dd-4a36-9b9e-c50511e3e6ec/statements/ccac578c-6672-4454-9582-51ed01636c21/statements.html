Alternate array

Your task is to write a short program that takes in input two arrays A and B composed by N numbers and gives in output an array C that contains A and B values in alternated way: First A element, Last B element, Second A element, Penultimate B element, ...

Input

The first line contains a number N that is the expected numbers to take in input.
The second line contains all the N numbers divided by space “ “ of the array A.
The third line contains all the N numbers divided by space “ “ of the array B

Output

The output contains the array C.

4
