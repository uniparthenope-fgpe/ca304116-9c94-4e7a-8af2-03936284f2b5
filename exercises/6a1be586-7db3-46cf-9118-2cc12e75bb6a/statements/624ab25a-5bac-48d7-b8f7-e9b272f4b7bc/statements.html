All Lowercase!

Write a function in C, called alltolower, which converts all characters of the string passed as a parameter to lowercase. In practice, this is the "enhanced" version of the tolower library function, which however only acts on a single character.

Input

The first line contains the string A.

Output

The output contains the lowercase string A.

Sample Input Sample Output
Hello hello
FGPE fgpe
GaMing gaming
