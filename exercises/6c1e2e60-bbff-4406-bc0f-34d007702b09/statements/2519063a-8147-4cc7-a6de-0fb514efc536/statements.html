Railway

On a railway line, compared to the full rate, Retired users benefit from a discount of 10%, Students by 15% and the Unemployed by 25%.
By coding retirees with a P, students with an S and unemployed with a D, write a program that, required the cost of a ticket and any condition particular user, see the amount to be paid.

Input

The first line contains the amount to be paid.
The second line contains the category (P,S,D) or none

Output

The first line contains the total with the respective class discount.

Sample Input Sample Output
35,00
S		 29,75
35,00
T		 35,00
180,50
D		 135,38
